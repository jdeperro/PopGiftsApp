## Design System

This directory contains design assets, specifications, and exported components from Figma.

### Directory Structure
- `/screens` - Exported screen designs
- `/components` - Individual component exports
- `/assets` - Icons, images, and other assets
- `/specs` - Design specifications (colors, typography, spacing)

## Usage

These design files inform the implementation of the Digital Gift Card Platform frontend, particularly:
- Card creation and editing interface
- Template gallery
- Checkout and payment screens
- Recipient card reveal experience
- Group gifting orchestration UI
